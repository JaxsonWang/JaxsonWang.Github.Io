对Android的一些折腾笔记

发表于 2016-08-19 | 更新于 2018-09-29 | 分类于 奇淫技巧 |

前言

今天莫名其妙想看看一加3的分区，尝试使用的之前的命令去测试，发现貌似都失效了。

以下都是以前的笔记

查看android分区列表

cat /proc/partitions
ls -al /dev/block/platform/msm_sdcc.1/by-name

查看分区详细信息

adb make_ext4fs /dev/block/mmcblk0p14

导出镜像文件

dd if=/dev/block/mmcblk0p14 of=/sdcard/system.img

dat文件转换img镜像

./sdat2img.py system.transfer.list system.new.dat system.img

挂载img镜像到输出文件夹

sudo mount -t ext4 -o loop system.img system/

打包镜像文件 一加(1.4G)

./makeext4fs -s -l 1388314624 -a system newsystem.img system/

打包后img镜像非4ext格式需要转换：

simg2img newsystem.img systemout.img

img转换dat

./rimg2sdat system_out.img
