好了，密码默认是空的，到目前来看，安装好了，本地测试也不需要配置太多的东西。

mysql -u root -p



//启动数据库服务
mysql.server start
//重启数据库服务
mysql.server restart
//停止数据库服务
mysql.server stop

安装phpMyAdmin

直接说配置了：将phpmyadmin/libraries下的config.default.php，把它复制到/phpmyadmin并重命名为config.inc.php

打开修改：

$cfg['PmaAbsoluteUri'] = 'http://localhost/phpMyAdmin';

允许空密码登录：

$cfg['Servers'][$i]['AllowNoPassword'] = true;

登录出现#2002报错：

$cfg['Servers'][$i]['host'] = '127.0.0.1';
